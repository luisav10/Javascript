If else (se/senão)

A condicional if é uma estrutura condicional que executa a afirmação, dentro do bloco, se determinada condição for verdadeira. Se for falsa, executa as afirmações dentro de else.

Exemplos de uso

O if-else é usado para fazer o seu programa reagir de forma diferente a situações diferentes.

  • Tomar decisões: O programa escolhe o que fazer. Por exemplo, "SE chover, levo guarda-chuva; SENÃO, não levo."
  • Mudar o que acontece: A página muda o jeito de agir. "SE o usuário clicar, mostro algo; SENÃO, escondo."
  • Comandar a ordem: Define qual pedaço do código vai funcionar. Isso faz a página se adaptar e responder bem a tudo (responsividade).

Switch case (escolha/caso)

O switch é ideal quando você tem uma única expressão (como o valor de uma variável) e quer executar diferentes blocos de código com base em múltiplos valores possíveis dessa expressão. É uma alternativa mais limpa e legível ao if-else aninhado para cenários específicos.

Exemplos de uso

Use o switch quando seu código precisa escolher uma ação de uma lista de opções fixas, baseada em um único valor. É perfeito para organizar lógicas onde "se for isso, faça X; se for aquilo, faça Y"

  • Tipo de Usuário: Diferentes ações para "Admin", "Editor" ou "Leitor".
  • Dias da Semana: Mostrar o nome do dia com base em um número (0 a 6).
  • Opções de Menu: Ativar funções distintas para "Home", "Perfil" ou "Configurações".

For (para)

É uma estrutura de controle de loop usada para executar um bloco de código repetidamente. É ideal para quando você sabe quantas vezes quer que o loop seja executado, como ao percorrer elementos de um array.

Exemplos de uso

Use sempre que a sua aplicação precisar fazer a mesma coisa várias vezes, de forma controlada e com um ponto de partida e chegada.

  • Exibir Produtos: Percorrer um array de produtos para mostrar o nome e preço de cada um em um e-commerce.
  • Contagem Regressiva: Fazer um timer de 10 a 0.
  • Animar Elemento: Mudar a posição de um objeto na tela 50 vezes para simular um pequeno movimento.

While (enquanto)

A estrutura while em JavaScript executa um bloco de código repetidamente enquanto uma condição for verdadeira. Ela é útil quando o número exato de iterações é desconhecido, mas o critério de parada é conhecido.

Exemplos de uso

Use o while quando precisar repetir uma ação enquanto uma condição for verdadeira, sem saber quantas vezes isso acontecerá. Ele continua executando o código até que a condição mude.

  • Esperar Entrada: Repetir a pergunta ao usuário até ele digitar uma senha correta.
  • Rolagem Infinita: Carregar mais conteúdo na página enquanto o usuário rola para baixo.
  • Processar Fila: Tirar itens de uma lista um por um até que a lista esteja vazia.

Array

Um array em JavaScript é uma estrutura de dados que armazena múltiplos valores em uma única variável, organizados como uma lista. Cada item no array é acessado por um índice numérico que começa em 0.

Exemplos de uso

Arrays são como listas organizadas que guardam vários itens em uma única "caixa". Você os usa sempre que precisa armazenar uma coleção de informações.

  • Imagens da Galeria: Manter a sequência de URLs de fotos para exibir em uma galeria.
  • Lista de Compras: Guardar todos os itens que o usuário adicionou ao carrinho.
  • Nomes de Alunos: Armazenar uma coleção de nomes de estudantes em uma turma.

Function

Funções são blocos de código que realizam tarefas específicas e podem ser reutilizadas ao serem chamadas pelo seu nome seguido por parênteses, o que as torna fundamentais para organizar e reutilizar o código.

Exemplos de uso

Use-as para organizar e reutilizar código, evitando repetições. Funções deixam seu programa mais limpo e fácil de entender.

  • Mudar Tema: Uma função que alterna entre o tema claro e escuro da página ao ser clicada.
  • Validar Formulário: Uma função que checa se todos os campos de um formulário foram preenchidos corretamente.
  • Calcular Média: Uma função que recebe notas e retorna a média final de um aluno.
