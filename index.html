If else

A condicional if é uma estrutura condicional que executa a afirmação, dentro do bloco, se determinada condição for verdadeira. Se for falsa, executa as afirmações dentro de else.

Exemplos de uso

O if-else é usado para fazer o seu programa reagir de forma diferente a situações diferentes.

  • Tomar decisões: O programa escolhe o que fazer. Por exemplo, "SE chover, levo guarda-chuva; SENÃO, não levo."
  • Mudar o que acontece: A página muda o jeito de agir. "SE o usuário clicar, mostro algo; SENÃO, escondo."
  • Comandar a ordem: Define qual pedaço do código vai funcionar. Isso faz a página se adaptar e responder bem a tudo (responsividade).

Switch case

O switch é ideal quando você tem uma única expressão (como o valor de uma variável) e quer executar diferentes blocos de código com base em múltiplos valores possíveis dessa expressão. É uma alternativa mais limpa e legível ao if-else aninhado para cenários específicos.

Exemplos de uso

Use o switch quando seu código precisa escolher uma ação de uma lista de opções fixas, baseada em um único valor. É perfeito para organizar lógicas onde "se for isso, faça X; se for aquilo, faça Y"

  • Tipo de Usuário: Diferentes ações para "Admin", "Editor" ou "Leitor".
  • Dias da Semana: Mostrar o nome do dia com base em um número (0 a 6).
  • Opções de Menu: Ativar funções distintas para "Home", "Perfil" ou "Configurações".
